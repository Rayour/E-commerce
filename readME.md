# Обучающий проект для изучения ООП на Python

## Описание:

Проект представляет из себя бекенд часть интернет магазина.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Rayour/E-commerce.git
```
2. Установите зависимости:
```
poetry instal 
```
