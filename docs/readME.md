# Проект E-commerce

## Пакет src

Содержит основные функциональные модули

### Модуль category

Содержит описание класса для категорий товаров с полями:
* имя
* описание
* список товаров
```python
name: str
description: str
products: list
```

А также атрибутами класса для подсчета:
* количества категорий
* количества товаров
```python
category_count = 0
product_count = 0
```

#### Функция инициализации __init__(self, name: str, description: str, products: list | None = None) -> None

Позволяет создать объект категории товаров с продуктами или без них

#### Метод __str__(self) -> str:

Метод строкового представления объекта категории:

Бытовая техника, количество продуктов: 56 шт.

В качестве количества продуктов возвращается суммарное количество всех продуктов данной категории

### Модуль product

Содержит описание класса для товаров с полями:
* имя
* описание
* цена
* количество
```python
name: str
description: str
price: float
quantity: int
```

#### Функция инициализации __init__(self, name: str, description: str, price: float, quantity: int) -> None

#### Метод __str__(self) -> str:
Метод строкового представления объекта продукта

Шоколад, 120.0 руб. Остаток: 55 шт.

#### Метод __add__(self, other: Any) -> float

Метод сложения продуктов. Возвращает суммарную стоимость всех единиц товаров в складываемых продуктах.
Если в качестве аргумента передается не объект класса Product - вызывается исключение
```
TypeError("Сложить можно только два объекта класса Product")
```

Позволяет создать объект товара

### Модуль utils

Содержит вспомогательные функции для обработки данных

#### Функция read_json(file_path: str) -> list

Функция получает путь до json файла,
возвращает список словарей с данными о категориях и продуктах

**Пример**:
```
print(read_json('excample.json'))

>>>[

        "products": [
            {
                "name": "55\" QLED 4K",
                "description": "Фоновая подсветка",
                "price": 123000.0,
                "quantity": 7
            }
        ]
    }
]
```

#### Функция create_categories(categories_list: list[dict]) -> list[Category]

Функция получает на вход список словарей с данными о категориях товаров и товарами в них,
возвращает список объектов категорий с вложенными объектами товаров

### Модуль product_iterator

#### Класс ProductIterator

Содержит описание класса для итератора товаров в категории:
* категория
* индекс
```python
category: Category
index: int
```
Позволяет производить итерации по продуктам категории
